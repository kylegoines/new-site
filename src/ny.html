New York Times - Events

Future-forward Catalogue

Brief

The New York Times wanted a unique application experience to manage and track past and future events. They wanted users to be able to filter through existing tags as well as be able to extend custom tags and taxonomies as the site grew. By building a custom experience using laravel, tailwind and modern javascript we were able to deliver a design system and UI system that lives within the New York Times brand, but with a clean modern interface.

The system is backed with a storybook pattern library for standalone UI elements and a CI/CD pipeline of redundancy checks to confirm the stability and expected behavior of the interface.
